Wyse Wiki

This website is intended as a tool to document the information relevant to the Wyse household. It will contain information on the network infrastructure, hobbies, calendars, links to user manuals for household products, pictures, backup support, and any other relevant information.

Network Information

  • Network Information
    • Hardware Layout & Devices -  Provides a network diagram, documents the current network devices, uses, and interconnections
    • Network Setup - Describes how to initially create the network
    • Network Services - Reviews the existing services, their use, setup and configuration
    • Network Backup - Describes how the network components and applications are backed up, and how to restore them if necessary
  • Household
    • Pictures
    • User Manuals
    • Health Information
