This page currently contains pages for each intended server/application.  It would be better to have all information for an application/server on the same page or related pages.  This page could be modified to show the sequence of the setup, using a list of tasks.  These tasks would be direct links to anchors in the application/server page's initial setup section.

Rules

  1. List of Servers & Services
    1. This Wiki should contain a list of all servers used on the network
    2. Each server should have its software inventory available on this Wiki
      1. Preferably this would be automated, but information would be here,even if entered manually
      2. Software inventory should contain:
        1. OS Type & Version
        2. Installed packages and versions
        3. Hardware type & identifiers
        4. Network hardware & interfaces
      3. Each of these servers & software should have links on how to configure them
    3. Each server should its hardware inventory available on this Wiki
  2. As services/servers are added this Wiki will be updated
  3. Backups to another machine will be scheduled for all configuration and data
  4. Monthly restore operations will be scheduled to test backup
  5. Docker will be used for all services if possible
