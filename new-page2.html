Configure Ubiquiti Dream Machine for Multiple VLANs and Wi-Fi Networks

This guide provides detailed instructions on how to configure the Ubiquiti Dream Machine (UDM) to support multiple VLANs and associated Wi-Fi SSIDs to segment network traffic for management, user devices, IoT devices, guest access, and streaming services.

Prerequisites

  • Ubiquiti Dream Machine (UDM)
  • Ubiquiti Access Points (APs) compatible with UDM
  • Network cables
  • Access to UniFi Network application (latest version)

1. VLAN Configuration

Begin by configuring the VLANs on your UDM. VLANs help in segmenting the network which enhances security and improves network management.

Step 1: Access the UniFi Network Application

  1. Connect your device to the network managed by the UDM.
  2. Open a web browser and navigate to https://unifi.ui.com or the IP address assigned to your UDM.
  3. Log in using your Ubiquiti account credentials.

Step 2: Create VLANs

Navigate to the settings area and select 'Networks'. Click on 'Create New Network' and configure each VLAN as follows:

VLAN Name VLAN ID Purpose
WYSECHOICE_MGMT 10 Management
WYSECHOICE 40 User devices
WYSECHOICE_IOT 20 IoT Devices
WYSECHOICE_GUEST 50 Guest Access
WYSECHOICE_STREAMING 30 Streaming Devices

For each network:

  • Enter the Name and VLAN ID.
  • Configure DHCP settings as required. Itâ€™s recommended to enable DHCP and set an appropriate range that does not overlap with other VLANs.
  • Save and apply changes.
