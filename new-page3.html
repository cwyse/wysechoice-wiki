Other Notable Settings

  • Domain name for DHCP is set to wysechoice.net on all networks
  • Internet access is enabled on the WYSECHOICE_DEFAULT, WYSECHOICE_IOT and WYSECHOICE_STREAMING networks
  • DHCP conflict checking is enabled on all networks except Primary (WAN1)
  • Auto scaling is disabled on all networks
  • IPv6 settings:
    • IPv6 setting preference is set to "auto" on all networks
    • IPv6 interface type is set to "none" on all networks
    • IPv6 PD start/stop is set to ::2 and ::7d1 on all networks
    • IPv6 client address assignment is set to SLAAC on all networks
    • IPv6 RA is enabled with a preferred lifetime of 14400 seconds on all networks
    • IPv6 RA priority is set to "high" on all networks
    • DHCP for IPv6 allows SLAAC on all networks
    • DHCP for IPv6 lease time is set to 86400 seconds on all networks
    • DHCP for IPv6 start/stop is set to ::2 and ::7d1 on all networks
    • DHCP for IPv6 DNS is set to automatic on all networks
  • IGMP proxy downstream is disabled on the WYSECHOICE_DEFAULT and Primary (WAN1) networks
  • IGMP proxy upstream is disabled on the Primary (WAN1) network
  • WAN settings on the Primary (WAN1) network:
    • WAN type is set to DHCP
    • IPv6 WAN type is disabled
    • WAN DNS preference is set to automatic
    • WAN load balancing type is set to "failover-only"
    • WAN SmartQueue is disabled with 10000 kbps up and 200000 kbps down
    • WAN provider capabilities are set to 250000 kbps down and 10000 kbps up
    • WAN events are set to be reported
